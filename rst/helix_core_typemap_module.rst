Synopsis

.. Description

- The typemap table associates file type modifiers with file patterns.
- This module manages the entire typemap table as a unit.
- This module supports check mode.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    # Set typemap entries
    - name: Configure typemap
      ripclawffb.helix_core.helix_core_typemap:
        state: present
        typemap:
          - type: binary+l
            path: //depot/....exe
          - type: binary+l
            path: //depot/....dll
          - type: text+k
            path: //depot/....txt
        server: '1666'
        user: bruno
        charset: auto
        password: ''

    # Clear all typemap entries
    - name: Clear typemap
      ripclawffb.helix_core.helix_core_typemap:
        state: absent
        server: '1666'
        user: bruno
        charset: auto
        password: ''
